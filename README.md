# Курсовая работа

> Тема: Разработка системы мгновенного обмена сообщениями с поддержкой видеоконференций

## Запуск сервера

Активация виратульного окружения:
```
. .venv/bin/activate
```

Установка зависимостей:
```
pip install -r app/requirements.txt
```

Запуск сервера:
```
./scripts/start-server.sh
```
